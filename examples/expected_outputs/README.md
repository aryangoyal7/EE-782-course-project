# Expected Outputs

This directory contains example visualizations that should be generated by the pipeline.

## Files

- `sample_input.png`: Example input image
- `predicted_mesh.png`: Rendered predicted 3D mesh
- `ground_truth_mesh.png`: Rendered ground truth mesh
- `volume_scatter.png`: Scatter plot of predicted vs ground truth volumes
- `pca_embedding.png`: PCA visualization of feature embeddings

## Generating Outputs

To generate these visualizations, run:

```bash
# After training
python -m src.training.evaluate \
    --config configs/default.yaml \
    --checkpoint checkpoints/best_model.pth \
    --output_dir examples/expected_outputs

# Or use the visualization scripts
python -m src.inference.visualize
```
